管理岗是什么鬼？

回想一下，你想要在公司里做管理的想法是从什么时候开始的？是什么让你产生想要做管理岗的想法的呢？

大学时工商管理课程？看到现在公司的里的管理层工作又轻松，拿钱又多？周围有做管理的朋友，他让你觉得你不如他？

各种MBA的招生广告，不知不觉植入你的脑海里的？

电视广告里那些叱诧风云运筹帷幄的精英形象？

某个电视剧里的人物？

总之，这个社会环境有很多信息源的输入，让你感觉不做管理，就低人一等，就是个职场菜鸟，凡职业规划最后都得到C字头的职位。但是否真的要做管理岗，那是得好好想一想的。

管理岗可能不是你想象的那么美好。首先，管理岗的工作压力更大。管理岗的工作内容庞杂，以中层管理为例，往往涉及到团队招募、成员培养、激励、职业规划，团队文化建设，涉及到业务层面的任务达成，部门协调（内部客户以及外部客户），发展规划，成本控制，行业动态洞察等多个方面。每个方面的工作都要消耗时间和精力，再加上个人成长和生活上的需要，不感觉到压力，那几乎是不可能的。

其次，管理岗的竞争压力也大。大部分公司都是金字塔结构的，造成了越在行政级别上高，职位数量越少。所以晋升难，而跳槽也难。

第三，可能薪资也没你想像的那么高。有些销售类的职位，基层销售的薪资可能比他们的经理，或者总监的高，因为有提成部分。有些替代性低的职位，薪资也是往往比他们的汇报对象高的。这是纵向来比。横向来比，有些行业内中利润水平一般的公司的管理岗的薪资，可能还不如行业中利润水平高的基层薪资。而有些公司，同一个公司内，技术性岗位的薪资完全可以匹敌管理岗，比如阿里的P线和M线。

管理岗可能不适合你

你平常工作的时候，是喜欢专心做事，没人打搅吗？管理岗不是这样的，可能时不时有人来找你，可能是老板，下属，客户，合作伙伴，旁部门。

还有你喜欢承担更大责任吗？下属犯错了，他可能一点事儿没有，但责任要算到管理者头上。

你是怎么来计算自己的成就的？是自己亲手做出来的让你更有成就，还是事情是你下属做的，你只是协助他而已，这样让你更有成就？

你愿意承担更大的不确定性吗？管理岗竞争激烈，压力又大，如果KPI不好，那可能换掉的时候管理者，而不是底层做事儿的人。

你具备说服力吗？管理岗对说服力的要求可比其他岗位要高。除了说服力之外，管理岗涉及的每个方面都必须要有高水准，有短板的管理者都会比较平庸。相对于市场上的众多竞争者，管理岗的人需要在管理岗所涉及的各方面具备突出的技术性水准。每个管理岗的职责貌似都差不多，但绩效差距可能是个鸿沟。那些没有经过足够训练就登上管理岗的人，和那些水到渠成，或者因为条件具足而获得系统训练的管理者比较起来，差别是巨大的，而待遇当然也是巨大的了。大部分人的工作都只是训练了这个人某一个或者几个方面的技术性能力，而管理岗是一个要求更多方面复合能力的岗位。所以，最好的办法是假以时日。有些公司会给年轻的员工轮岗机会，但不见得最后这些人能成为成功的管理者。因为除了能力之外，还有其他软性的东西，比如个性、价值观、周围环境等各种原因来造就一个成功的管理者。

绝大多数想做管理岗的职场人，可能是被误导的。市场容纳不了那么多管理者。职能上的技术性深度和广度才是市场所稀缺的。对于管理岗的关注，可能可以作为逃避持续在不那么激动人心的日常工作上的继续深入，也可以用来逃避认识什么样的工作才是适合自己的。而在努力转向管理岗的过程中，耗费了提升技术性能力的机会，也降低了自身的竞争力。

到管理岗的最好的路径是水到渠成，保持管理岗的方向，但不拔苗助长。
