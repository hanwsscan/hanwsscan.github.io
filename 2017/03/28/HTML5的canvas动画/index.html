HTML5的canvas动画

HTML5 canvas 标签用于绘制图像，canvas 元素本身并没有绘制能力（它仅仅是图形的容器） 必须使用脚本（通常是 JavaScript）来完成实际的绘图任务。

Internet Explorer 9 +、Firefox、Opera、Chrome 以及 Safari 支持 canvas 及其属性和方法。

getContext() 方法可返回一个对象，该对象提供了用于在画布上绘图的方法和属性。

一、标签原型

canvas fillStyle 属性，绘制矩形样式

canvas strokeStyle 属性，绘制边框的样式

canvas shadowColor 属性，绘制阴影的样式

canvas lineCap 属性，绘制圆形的结束线
